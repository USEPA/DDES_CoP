Install postman: https://www.getpostman.com/apps

Import the JS file representing the Postman collection:
In postman click 'Import File' (top left)
Select the DDES.postman_collection.json file from your machine and click Import

In the newly imported DDES collection (left column) select the API request you wish to execute by clicking
A tab will open representing that request
Click 'Send' from that tab and see the results
